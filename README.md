# Halal Finance Knowledge Base

Персональная база знаний по исламским финансам, халяльным инвестициям и фикху муамалят (исламского права в сфере финансовых транзакций).

---

## Назначение

Этот репозиторий — структурированная база знаний для использования совместно с Claude через Anthropic Projects. Claude обращается к файлам из этого репозитория при ответах, чтобы давать персонализированные ответы, основанные на:

- **Ханафитском мазхабе** как приоритетной правовой школе
- **Казахстанском контексте** (ДУМК, AIFC/МФЦА, ADCB Islamic Bank, KASE, тенге, налоговая система РК)
- **Международных шариатских стандартах** (AAOIFI, IFSB)
- **Современной академической литературе** по исламским финансам

Репозиторий **не является** общедоступным учебником, фетва-сервисом или шариатской консультацией. Это персональный инструмент.

---

## Как это работает

```
┌─────────────────────────┐         ┌──────────────────────────┐
│   Claude Project        │ ──────> │   GitHub (этот репо)     │
│   (Project Instructions │         │                          │
│    + Project Knowledge) │  fetch  │   - rules.md             │
│                         │ <────── │   - INDEX.md (роутер)    │
└─────────────────────────┘         │   - skills/*.md          │
           ▲                        │   - sources/*.md         │
           │                        └──────────────────────────┘
   Личные данные:
   - my-portfolio.md
   - my-budget.md
   (только в Project Knowledge,
    не в публичном репо)
```

1. В Claude Project настроены инструкции, которые ссылаются на raw-URL файлов этого репозитория.
2. При вопросе пользователя Claude через `web_fetch` подгружает нужный `SKILL.md` из репозитория.
3. Личные финансовые данные пользователя хранятся в Project Knowledge (загруженные файлы внутри проекта) и **никогда не попадают в этот публичный репозиторий**.

## Иерархия источников

При наличии расхождений между источниками применяется следующий приоритет:

1. **Ханафитский мазхаб и фетвы ДУМК** — для практических решений в Казахстане
2. **Шариатские стандарты AAOIFI** — для финансовых инструментов
3. **Стандарты IFSB** — для регулирования и риск-менеджмента
4. **Современные ханафитские учёные** (Таки Усмани и др.) — для современных вопросов
5. **Другие мазхабы** (шафии, малики, ханбали) — для справки и расширенного понимания

Принцип «не выбирать удобное мнение» (запрет тальфика и татабу' ар-рухас) явно прописан в `rules.md`.

## Дисклеймер

- Содержание репозитория **не является шариатской фетвой**.
- Не заменяет консультацию с квалифицированным шариатским советником.
- Не является лицензированной финансовой рекомендацией.
- Финальное решение по любому финансовому или религиозному вопросу остаётся за пользователем после консультации с квалифицированными специалистами.
